Conditionals

Conditionals allow your code to make decisions and execute different blocks of code based on whether a condition is true or false. This is fundamental for controlling the flow of your program.

Example 1: Even or Odd Check (`if/else`)

This example uses a simple if and else statement to check if a number is even or odd based on the remainder when divided by 2.

Example 2: Positive, Negative, or Zero (`if/else if/else`)

This example demonstrates using if, else if, and else to check for multiple possible conditions.

Example 3: Checking a Range (Logical Operators `&&`, `||`)

Logical operators like && (AND) and || (OR) allow you to combine multiple conditions. This example checks if a number is between 10 and 20 (inclusive) using the AND operator.
