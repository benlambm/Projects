Arrays

Arrays are ordered lists of values. They are fundamental data structures in JavaScript, allowing you to store and manage collections of data.

Example 1: Creating and Accessing Elements

Arrays are created using square brackets []. You can access individual elements using their index, starting from 0 for the first element.

Example 2: Adding Elements (`push`, `unshift`)

Use the push() method to add an element to the end of an array, and unshift() to add an element to the beginning.

Example 3: Removing Elements (`pop`, `shift`)

Use the pop() method to remove the last element from an array, and shift() to remove the first element.

Example 4: Iterating Through an Array (for loop)

You can loop through array elements using a for loop and the array's length property.

Example 5: Transforming Arrays (`map`, `filter`)

Higher-order functions like map() and filter() are powerful ways to create new arrays based on existing ones. This example doubles each number in an array.
