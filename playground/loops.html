Loops

Loops are used to repeat a block of code multiple times. They are essential for tasks that involve iteration, like processing lists of data or performing actions a set number of times.

Example 1: For Loop (Multiplication Table)

The for loop is often used when you know in advance how many times you want the loop to run. This example generates a multiplication table for the number you enter.

Example 2: While Loop (Counting Up)

The while loop is used when you don't know exactly how many times the loop will run, but you want it to continue as long as a specific condition is true. This example counts up from 1 to the number you enter.
