DOM (Document Object Model)

The DOM is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. JavaScript interacts with the DOM to make web pages dynamic.

Example 1: Selecting Elements

You can select HTML elements in your page using methods like getElementById() (by ID), querySelector() (by CSS selector, selects the first match), and querySelectorAll() (by CSS selector, selects all matches).
