Objects

Objects in JavaScript are collections of key-value pairs, where keys are strings (or Symbols) and values can be any data type, including other objects or functions. They are used to represent entities and group related data and functionality.

Example 1: Creating and Displaying an Object

Objects can be created using curly braces {}. Properties are defined as key: value pairs. Enter a name and age to create a simple person object.

Example 2: Accessing Properties

You can access object properties using dot notation (object.property) or bracket notation (object['property']). Bracket notation is useful for keys with spaces or special characters, or when the key is stored in a variable.

Example 3: Modifying Properties

You can change the value of an existing property by assigning a new value to it using either dot or bracket notation.

Example 4: Adding New Properties

You can add a new property to an object by assigning a value to a new key using dot or bracket notation.

Example 5: Deleting Properties

You can remove a property from an object using the delete keyword.

Example 6: Iterating Through Properties

You can loop through the keys (and thus the values) of an object using a for...in loop or methods like Object.keys(), Object.values(), or Object.entries().
